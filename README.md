## ✦ Proje Vizyonu & Mimari

Bu depo, sametcolak.com.tr adresinde yayında olan kişisel portfolyo platformumun kaynak kodlarını barındırmaktadır. Geleneksel şablonların ötesinde; **modern web standartlarına uygun, performans odaklı ve SEO uyumlu** bir mimariyle sıfırdan, hiçbir kütüphane kullanılmadan tasarlanmıştır.

### ✨ Öne Çıkan Özellikler

- 🚀 **Zero-Dependency (Sıfır Bağımlılık):** React, Bootstrap veya Tailwind gibi dış çerçeveler (framework) kullanılmadan, tamamen **Vanilla JS** ve **saf CSS** ile geliştirilmiş anında yüklenen altyapı.
- 🎨 **A-Kalite Modern UI/UX:** 
  - Gelişmiş **Glassmorphism** (cam efekti) ve saydam arka planlar (backdrop-filter).
  - Modern asimetrik **Bento Grid** (kutu) düzenleri.
  - Özel olarak tasarlanmış interaktif 3D animasyonlar ve yumuşak sayfa geçişleri.
- 🌍 **Dinamik Dil Desteği:** Kullanıcı deneyimini bölmeden, anlık olarak sayfa yenilenmeden çalışan **İngilizce / Türkçe** dil çeviri sistemi.
- ⚡ **Akıllı Optimizasyonlar:** Özel donanımsal imleç (Custom Cursor), Scroll Fade-In belirmeleri, akıllı Preloader ve performansı yormayan CSS keyframe animasyonları.
- 🔍 **Erişilebilirlik & SEO:** Özelleştirilmiş `sitemap.xml`, `robots.txt` entegrasyonları, semantik HTML5 standartları ve tam duyarlı (%100 Responsive) tasarım kurgusu.

